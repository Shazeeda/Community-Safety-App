🚨 Community Safety App
The Community Safety App is a full-stack web application that empowers users to report, view, update, and delete local incidents in their community. It supports user authentication, protected routes, and stores data in a PostgreSQL database.

📌 Features
🔐 User Authentication – Secure login and registration using JWT tokens and hashed passwords.

📝 CRUD Functionality – Users can Create, Read, Update, and Delete incidents.

📚 Protected Routes – Access to certain pages requires a valid token.

📦 PostgreSQL Database – Stores all users and incidents.

🧪 Seeded Data – Pre-loaded sample incidents for easy testing.

⚠️ Error Handling – Graceful handling of form errors and API responses.

🎨 Responsive UI – Clean and user-friendly layout with CSS styling.

🧭 React Router – Handles client-side routing between pages.
